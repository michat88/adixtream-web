Tentang Aplikasi

Apa itu AdiXtream?

Kami memahami bahwa tidak semua pengguna memiliki waktu untuk melakukan konfigurasi manual. Oleh karena itu, AdiXtream hadir dengan konfigurasi repositori yang sudah terintegrasi secara otomatis dan antarmuka yang lebih sederhana. Dengan ini, Anda bisa langsung menikmati konten pilihan segera setelah aplikasi terpasang di perangkat Anda.

Cara Pembelian & Aktivasi

  1. 1

    Buka Aplikasi & Salin Device ID

    Masuk ke pengaturan Extensions/Plugins, sebuah popup Premium akan muncul. Ketuk pada DEVICE ID ANDA untuk menyalin ID tersebut.

  2. 2

    Lakukan Pembayaran (QRIS)

    Scan kode QRIS yang tersedia di dalam aplikasi menggunakan e-wallet (OVO, DANA, GoPay, ShopeePay) atau M-Banking Anda sesuai paket yang dipilih.

  3. 3

    Hubungi Admin di Telegram

    Kirimkan Bukti Pembayaran beserta Device ID Anda ke Admin Telegram kami untuk mendapatkan Kode Unlock.

  4. 4

    Masukkan Kode Unlock

    Kembali ke aplikasi AdiXtream, tempelkan Kode Unlock yang diberikan Admin, dan klik UNLOCK NOW. Aplikasi akan direstart dan Premium aktif!

Cara Instalasi

  1. Unduh file APK terbaru melalui tombol Download di atas.
  2. Buka file APK tersebut di perangkat Android Anda.
  3. Berikan izin instalasi dari "Sumber Tidak Dikenal" jika diminta oleh sistem.
  4. Jalankan aplikasi AdiXtream.
  5. Tunggu proses inisialisasi sejenak, dan selamat menonton!

Mode Aduh Ma Aduh Pa (NSFW)

Bagi Anda yang membutuhkan konten ekstra, fitur NSFW sudah tersedia dan dapat diaktifkan dengan langkah berikut:

  • Menu Penyedia: Buka Pengaturan > Penyedia, lalu aktifkan tombol "Mengaktifkan NSFW pada Ekstensi yang didukung".
  • Pilihan Media: Klik menu "Media yang lebih diinginkan" dan pastikan opsi NSFW sudah dicentang.
  • Filter Beranda: Kembali ke halaman utama, lalu aktifkan filter NSFW di barisan kategori bawah agar konten muncul.

Jalur Aman (Anti-Blokir ISP)

Agar akses tontonan Anda tidak terganggu oleh pemblokiran internet (Internet Positif), aktifkan fitur DNS:

Pergi ke Pengaturan > Umum. Aktifkan fitur DNS over HTTPS. Fitur ini sangat berguna untuk melewati blokir ISP dan menjaga koneksi tetap stabil saat melakukan scraping konten.
